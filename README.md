# 🖥️ Bash System Monitor

A lightweight bash script that monitors CPU, Memory, and Disk usage and warns when usage exceeds 80%.

## Usage
```bash
chmod +x monitor.sh
./monitor.sh
```

## Tech Stack
- Bash scripting
- Linux CLI tools (top, free, df)
